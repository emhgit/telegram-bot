## Usage

1. Running the Bot
    - Start the bot:
        ```bash
            node index.js
2. Keep Bot Running
    - Local:
        ```bash
            npm install -g pm2
            pm2 start index.js --name "telegram-bot"
    - Cloud:
        Use Render.com, Railway.app, or Replit (free options).
        Push code to GitHub, link to a cloud provider, and configure BOT_API_TOKEN as an environment variable.
